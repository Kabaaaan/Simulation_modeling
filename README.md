# Simulation Modeling

Репозиторий содержит решения лабораторных работ по курсу *имитационного моделирования*.
Все задания выполнены в среде **Google Colab** и посвящены применению стохастических методов, в первую очередь метода Монте-Карло и его расширений.

## 📌 Содержание

В репозитории представлены три лабораторные работы:

* **Лабораторная работа №1** — метод Монте-Карло и генерация случайных величин
* **Лабораторная работа №2** — моделирование распространения излучения и методы понижения дисперсии
* **Лабораторная работа №3** — фильтр частиц

---

## 🔬 Лабораторная работа №1

**Метод Монте-Карло и генерация распределений**

В рамках работы реализованы и исследованы базовые методы имитационного моделирования:

* Генерация случайной величины с заданной плотностью
* Проверка гипотезы согласия с теоретическим распределением
* Моделирование инвестиционного проекта:

  * оценка математического ожидания, дисперсии и коэффициента вариации NPV
* Оценка характеристик мощности ветрогенератора:

  * построение доверительных интервалов для различных распределений скорости ветра
* Моделирование случайных процессов:

  * анализ времени достижения порога
* Моделирование прохождения частиц через среду:

  * оценка потерь энергии

Основной акцент сделан на применении метода Монте-Карло для оценки характеристик сложных случайных систем.

---

## 💡 Лабораторная работа №2

**Методы понижения дисперсии и моделирование распространения излучения**

Работа состоит из двух частей:

### 1. Методы понижения дисперсии

* Оценка вероятности отказов системы
* Использование метода контрольных случайных величин
* Применение метода существенной выборки (importance sampling)
* Сравнение дисперсии оценок для различных подходов
* Ambient occlusion

### 2. Моделирование распространения фотонов

Реализована имитационная модель распространения излучения в многослойной биологической среде (модель глазного дна):

* Трассировка движения фотонов
* Учет:

  * поглощения
  * рассеяния (фазовая функция Хени–Гринштейна)
* Работа со статистическим весом фотонов
* Оценка:

  * распределения поглощенной энергии
  * отраженного излучения

Задача демонстрирует применение Монте-Карло в задачах биомедицинской оптики.

## 🤖 Лабораторная работа №3

**Фильтр частиц (Particle Filter)**

Реализован алгоритм фильтра частиц для задачи локализации мобильного объекта:

* Оценка состояния:

  * координаты
  * ориентация
  * неизвестный параметр среды
* Учет:

  * шумов управления
  * шумов измерений
* Использование вероятностной модели движения и наблюдений
* Ресемплинг частиц

Модель основана на задаче отслеживания движения объекта внутри многоугольной области по данным датчика расстояния до границы.
